# Jogo-da-cobrona
Esse é um projeto da faculdade feito em python e pygame pra praticar criando um "jogo da cobrinha" com algumas requisições feitos pelo professor Pedro Henrique.

- Segui os passos a passos do professor pra fazer a logica da cobra, comida e loop do jogo e fiz uma tela de gameover no mesmo estilo do jogo Undertale, em que tive que importar a fonte 8-Bit Wonder;
- Em seguida adicionei uma atualização opcional que precisaria no trabalho, então adicionei uma seleção de skins com 6 cores antes do inicio do jogo, que consegui deixar de uma forma bem simples com seleção numerica;
- Eu também havia adicionado uma regra pra corrigir um bug de colisão, em que ao ir pra direção oposta a que a cobra está andando, a cobra batia nela mesma então daria gameover. Então fiz uma lógica pra impedir que a cobra possa ir na direção oposta durante o percurso;


É um jogo simples em python mas que concerteza me ajudou a entender a logica de várias funções trabalhando juntas, e importação de fonte externas que eu ainda não havia precisado usar antes.
